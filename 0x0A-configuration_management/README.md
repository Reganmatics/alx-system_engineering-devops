```DevOps```, ```SysAdmin```, ```Scripting```, ```CI/CD```

install ```puppet```
```
$ apt-get install -y ruby=1:2.7+1 --allow-downgrades
$ apt-get install -y ruby-augeas
$ apt-get install -y ruby-shadow
$ apt-get install -y puppet
```

install puppet-lint
```
$ gem install puppet-lint
```
